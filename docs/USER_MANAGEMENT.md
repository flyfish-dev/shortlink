# 用户管理、公网登录与二维码样式

本版本在原管理员后台基础上增加公网用户能力。

## 角色

- `admin`：管理员，可查看全部短链、活码、用户和系统设置，可审核用户提交的短链与活码。
- `user`：普通用户，只能查看和维护自己创建的短链、活码和统计数据，不能访问用户管理、系统设置和审核接口。

## 公网用户登录

公网用户通过邮箱 Magic Link 快速登录。若邮箱尚未存在，系统会自动创建普通用户账户；Magic Link 仍为一次性链接，默认 15 分钟有效。

浏览器一键登录仍保留：已绑定浏览器可以快速进入；更换设备或清理 Cookie 后，可以通过 Magic Link 或恢复 Key 重新绑定。

## 审核流程

普通用户创建或修改短链、活码后，资源会进入 `pending` 状态。只有管理员审核通过后，公开短链跳转和活码展示才会实际生效。

## 资源隔离

后台 API 会按登录账户角色进行强制隔离：普通用户列表、详情、统计、编辑、删除都限制在自己的 `owner_account_id` 范围内；管理员不受该限制。

## 二维码样式

短链入口二维码和活码入口二维码支持：

- 经典方块：`classic`
- 圆角：`rounded`
- 圆点：`dots`
- 前景色：`qr_foreground`
- 背景色：`qr_background`

公开二维码接口 `/qr/short/{code}.png` 和 `/qr/live/{code}.png` 会根据资源配置动态渲染 SVG 二维码。
